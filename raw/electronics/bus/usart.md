Universal Asynchronous Receiver Transmitter
==============================

UART is serial bus.


Specification 
-------------

| Signals | |
|------|---|
| RX | Receiving Line (In) |
| TX | Transmitting Line (Out) |
| (RTS) | Request to Send |
| (CTS) | Clear to Send |

Note: TX signal of the sender is connected to the RX signal at the receiver!



Common Speeds
:   9'600 bit/s, 38'400 bit/s
